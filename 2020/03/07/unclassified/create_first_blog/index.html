2：新建一个Repository

1、创建一个 名为 username.github.io 的 Repository，username就是github的用户名。
2、例如 xiaowaijing.github.io， 访问地址就是 https://xiaowaijing.github.io

3、点击 Create repository 就OK了。

3：测试一下

1、点击 Create new file 新建一个 index.html 文件。

2、随便输入些字符，然后记得保存。

3、过一会儿访问就可以看到信息
